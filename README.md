# BDI SWEETS Artisanal Patisserie Platform

Welcome to the **BDI SWEETS** digital boutique—a luxury e-commerce platform specializing in traditional, artisanal Tunisian patisserie (Kaak Warka, Baklava, Samsa, Mlabes).

## Project Structure

This is a monorepo consisting of:
- **`frontend/`**: A React application built with Vite and Tailwind CSS, featuring a sophisticated redesign based on the "Baker" artisanal theme.
- **`backend/`**: A NestJS server providing the API for product management and order processing.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation & Development

#### 1. Frontend
```bash
cd frontend
npm install
npm run dev
```
The application will be available at `http://localhost:5173`.

#### 2. Backend
```bash
cd backend
npm install
npm run start:dev
```
The API server will run in watch mode at `http://localhost:3000`.

## Design System

The platform follows a premium artisanal aesthetic:
- **Primary Color**: Baker Signature Gold (`#EAA636`)
- **Accent Color**: Baker Dark Brown (`#1E1916`)
- **Background**: Baker Soft Cream (`#FDF5EB`)
- **Typography**: High-contrast Serif headings with elegant Sans-Serif body text.

## Features

- **Artisanal Product Catalog**: Categorized showcase of Tunisian sweets.
- **Heritage Storytelling**: About page detailing the 50-year mastery.
- **Service Highlights**: Dedicated sections for quality and artisanal preparation.
- **Responsive Checkout**: Secure and elegant ordering flow.
